# Editing the boards (KiCad 10+)

- **Main board + schematic**: open `project/Strampler_redesign_v2_2.kicad_pro`
  — a full linked project (schematic, board, and the Eagle-imported symbol
  library). Footprints are embedded in the board, so it is self-contained.
- **Panel**: open `strampler_panel_v2_2.kicad_pcb` directly in the PCB
  Editor. The artwork is ~34k filled polygons on F.Silkscreen; the copper
  pours are filled zones. It is heavy but edits fine.

After editing, regenerate all fab outputs (gerber zips + centroid) with:

```
./make-fab-files.sh
```

Both gerber sets passed PCBWay audit (2026-07) as generated by that script's
commands.

## Provenance & pitfalls

These files were converted from the Eagle 7.3 originals (`../hardware-eagle/`)
with KiCad 10.0.4. Two importer pitfalls are already fixed here — beware if
you ever re-import from Eagle instead of editing these files:

1. The importer creates **unfilled zones**, which plot as EMPTY gerber
   layers (missing copper, mask and artwork). Fixed by filling copper zones
   and converting non-copper zones to filled `gr_poly` shapes.
2. The importer maps Eagle's t/bRestrict layers to layer `"UNDEFINED"`,
   which KiCad's own parser then refuses to load. Remapped to `User.1`.

Verify against the Eagle originals and the `render-main-*.png` references
before fabbing modified boards. License: CC BY-NC-SA 4.0 (see ../LICENSE.md).
